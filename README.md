# GP2

A browser-based, low-poly, pseudo-simulator Formula 1 racing game set in the character of the 1990 F1 season: powerful, ground-effect-less, manual-gearbox cars that are hard to drive fast. Cockpit view, keyboard controls, flat-shaded/vertex-colored low-poly art (no textures). Built with Three.js (rendering) and Rapier (physics), in TypeScript via Vite.

See [docs/requirements.md](docs/requirements.md) for the full requirements doc and [plan/](plan/) for the phased build plan.

## Dev commands

```sh
npm install      # install dependencies
npm run dev      # start the dev server
npm run build    # type-check and build a production bundle to dist/
npm run preview  # preview the production build locally
```

## Deployment

Pushes to `main` build and deploy automatically to GitHub Pages via `.github/workflows/deploy.yml`.
